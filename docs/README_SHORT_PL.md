# Agent Control Plane - krotka wersja

## Co to jest

To operatorski cockpit do uruchamiania, przegladania i kontrolowania runow agentow AI.

## Co pokazuje

- Python i FastAPI
- runtime timeline
- policy checks i trust boundaries
- approvals i incidents
- replay compare
- eval scorecards

## Jak o tym powiedziec na rozmowie

To nie jest kolejny chatbot, tylko produktowa warstwa operatorska wokol agentow AI. Pokazuje, jak zespol moze przegladac zachowanie agentow, zatrzymywac ryzykowne akcje, porownywac replaye i pracowac na jednym cockpitcie review.

## Najmocniejsze elementy

- mission-control UX/UI
- structured traces
- compare mode
- queue ownership
- exporty markdown
